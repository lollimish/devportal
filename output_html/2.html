Request Parameters
Parameter
content-type string Header

Specifies the type of content of the body of the entity.Must be set to one of the following values:

  • application/xml

  • application/json


Note: If there is no entity bodyin a normal successful response, this parameter is still needed tospecify the format in the case of an error response message.

content-encoding string Header

Specifies the type of encoding used on the data. Supportedvalues are: identity or gzip


Note: If there is no entity bodyin a normal successful response, this parameter is still needed tospecify the format in the case of an error response message.
