0.0.1 Functional Behavior

Describe the functional behavior provided by this operation. Also add here any special considerations, assumptions, and dependencies for this particular operation
Add a list of file into an existing folder. Adding the same file twice in a folder is allowed. A folder “Antarctica” contains File1.txt
If a user selects 3 files, File1.txt, File2.txt, File3.txt and adds them to the folder “Antarctica”, folder “Antarctica” will contain, the following: File1.txt, File1.txt, File2.txt and File3.txt.

content-type: application/xml

accept: application/xml
