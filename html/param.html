Require Parameters
Parameter
content-encoding string Header

Specifies the type of encoding used on the data. Supported values are: identity or gzip

content-type string Header

Specifies the type of content of the body of the entity. Must be set to one of the following values:

  • application/xml

  • application/json

Note: If there is no entity body in a normal successful response, this parameter is still needed to specify the format in the case of an error response message.

location AnyURI Header

Specifies the location of the newly created resource. This URI can be used to interact with the newly created resource. See Get Album Contents for how the URI is structured.

album Object Body

Contains the album information.
