Add tracks to playlist
Overview

Insert a track into a playlist belonging to a user. Adding the sametrack twice in the same playlist is allowed. Considering the followingscenario: A playlist “Antarctica” contains Song1.mp3

If a user selects 3 tracks, Song1.mp3, Song2.mp3, Song3.mp3 and addsthem to the playlist “Antarctica”. Playlist “Antarctica” will contain,the following: Song1.mp3, Song1.mp3, Song2.mp3 and Song3.mp3.

  • OAuth Scope

  • Scope: LOCKER

  • Model: OAuth authorization_code

  • Resource
  • copy 
    /speechToText
