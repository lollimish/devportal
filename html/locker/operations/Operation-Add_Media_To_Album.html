Add media to album
Overview

Insert a media object or a collection of media objects into an albumbelonging to a user. Add the same Media twice in the same album isallowed. Considering the following scenario: An album “Antarctica”contains Photo1.jpg

If a user selects 3 pictures, Photo1.jpg, Photo2.jpg, Photo3.jpg andadds them to the album “Antarctica”. Album “Antarctica” will contain,the following: Photo1.jpg, Photo1.jpg, Photo2.jpg and Photo3.jpg.

  • OAuth Scope

  • Scope: LOCKER

  • Model: OAuth authorization_code
