Update folder
Overview

Modifies the information of a specific folder belonging to the specifieduser. If a user attempts to update a folder instance with name ofan already existing folder instance, service will throw an exception.

  • OAuth Scope

  • Scope: LOCKER

  • Model: OAuth authorization_code

  • Resource
  • copy
