Delete group
Overview

The Delete Group method enables deleting a list of groups. If an UID provided in the request is not valid, then this API will return an InvalidGroupException and no group will be removed. If the UID of the Favorites SYSTEM group is passed, this API will return an InvalidGroupException and no group will be removed.

Return:

  • If the call is successful, an empty envelope with an HTTP code of 204 will be returned.
  • If the subscriber number is incorrect or the subscriber’s account status is BLOCKED then return InvalidSubscriberException.
  • If an invalid group UID is present in the array, then return InvalidGroupException.
  • If any other error, then the SystemException will be returned.

Request Parameters
Parameter
authorization String Header

Specifies \textquotedbl{}Bearer\textquotedbl{} + OAuth Token – access\_token. If the Authorization header is missing, the system shall return an HTTP 400 Invalid Request message. If the token is invalid the system shall return an HTTP 401 Unauthorized message with a WWW-Authenticate HTTP header.

groupId String URI

Specifies group id to delete.
