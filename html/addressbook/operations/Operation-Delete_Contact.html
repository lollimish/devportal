Delete contact
Overview

Contact identified by the contactId are moved to the trashcan. myInfo’s contact id considered as an invalid contactId. CSAB does not use trashcan. Then, when a user deletes a contact, it must be purged from the system and it isn’t possible to restore it later.

Return:

  • If call is successful, an empty envelope with an HTTP code of 200.
  • InvalidSubscriberException if the sub_id is incorrect or the subscriber’s account status is BLOCKED.
  • InvalidContactException if one of the ContactId provided is invalid.
  • SystemException for any other error.

  • OAuth Scope

  • Scope: AAB

  • Model: OAuth authorization_code

Request Parameters
Parameter

Specifies textquotedblBearertextquotedbl + OAuth Token – access_token. If the Authorization header is missing, the system shall return an HTTP 400 Invalid Request message. If the token is invalid the system shall return an HTTP 401 Unauthorized message with a WWW-Authenticate HTTP header.
