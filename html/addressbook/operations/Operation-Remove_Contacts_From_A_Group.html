Remove contacts from a group
Overview

This method will remove the association between the specified group and list of contacts.

  • If the provided group’s id is invalid, this API will return an InvalidGroupException.
  • If one of the contactId in the array is invalid or is myInfo’s contact ID, this API will return an InvalidContactException and no association between the group and any contacts of the list will be removed.

Return:

  • If successful, an empty envelope with an HTTP code of 200.
  • InvalidContactException if a contact identifier is invalid in the provided list.
  • InvalidGroupException if the group identifier is invalid.
  • InvalidSubscriberException if the sub_id is incorrect or the subscriber’s account status is BLOCKED.
  • SystemException for any other error.

Request Parameters
Parameter
authorization String Header

Specifies textquotedblBearertextquotedbl + OAuth Token – access_token. If the Authorization header is missing, the system shall return an HTTP 400 Invalid Request message. If the token is invalid the system shall return an HTTP 401 Unauthorized message with a WWW-Authenticate HTTP header.

contactIds String Query String

Specifies list of “%2C” (,) delimited contact id’s to delete. Note: 20 group ids are allowed to delete.

Note: If no “contactIds” query parameter in the request, then remove all contacts from the group.

groupId String URI

Specifies the group identifiers to be deleted.
