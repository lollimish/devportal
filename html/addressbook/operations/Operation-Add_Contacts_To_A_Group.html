Add contacts to a group
Overview

This method associates a list of contacts to a group.

  1. If an invalid group UID is provided, this method will return an InvalidGroupException.
  2. If a contact already belongs to the specified group, nothing is changed for it.
  3. If an invalid contactId is provided, this method will return an InvalidContactException.
Return:
  • If successful, an empty envelope with an HTTP code of 204.
  • InvalidSubscriberException if the sub_id is incorrect or the subscriber’s account status is BLOCKED.
  • InvalidContactException if one of the contacts UID is not valid.
  • InvalidGroupException if the group UID is not valid.

Request Parameters
Parameter
authorization String Header

Specifies \textquotedbl{}Bearer\textquotedbl{} + OAuth Token – access\_token. If the Authorization header is missing, the system shall return an HTTP 400 Invalid Request message. If the token is invalid the system shall return an HTTP 401 Unauthorized message with a WWW-Authenticate HTTP header.

contactIds String Query String

Specifies list of “\%2C” (,) delimited contact id’s to add. Note: 20 group ids are allowed to add.

groupId String URI

Specifies the group identifiers to be deleted.

Require Parameters
Parameter
date Date Time Header

Specifies Date and time the response was returned (in RFC1123 format). For example Date: Sun, 06 Nov 1994 08:49:37 GMT
