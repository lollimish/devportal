Update myinfo
Overview

The Update MyInfo method updates a subscriber personal profile MyInfo based on the provided contact data structure. The same rules regarding the picture size and format are applied.

Return:

  • If the call is successful, it returns 204 status codes with empty response.
  • If the sub_id is incorrect or the subscriber’s account status is BLOCKED, then return InvalidSubscriberException.
  • If any other error, then return SystemException.

  • OAuth Scope

  • Scope: AAB

  • Model: OAuth authorization_code

Request Parameters
Parameter
accept-encoding String Header

Specifies accept encoding format

gzip,deflate

When this header is present, the gateway should compress the response to indicate the client that response is in compressed format

authorization String Header

Specifies \textquotedbl{}Bearer\textquotedbl{} + OAuth Token – access\_token. If the Authorization header is missing, the system shall return an HTTP 400 Invalid Request message. If the token is invalid the system shall return an HTTP 401 Unauthorized message with a WWW-Authenticate HTTP header.

content-type String Header

Specifies the representation format for the request. This is the content type for the whole message. The acceptable values for this parameter are:

• application/json

• application/xml
